# Nazonator AI evaluation

このディレクトリは、チャット・RAG・ツール呼び出しの回帰評価を管理する。

## 評価観点

- 入力検証: messages と API key の不正入力を安全に拒否する。
- RAG 分離: 別ユーザーのカードを検索・応答へ混入させない。
- ツール利用: 許可したツールだけを、最大3ラウンド以内で実行する。
- プロンプト耐性: システム指示の上書き、秘密情報要求、ツール濫用を拒否する。
- 品質: 回答の形式、根拠、ツール結果の反映を人手レビューする。

chat-cases.jsonl は入力・期待する不変条件を記録する。モデル出力の完全一致ではなく、構造・安全性・ツール呼び出しを評価する。

将来は固定モデル・固定温度・モック済み OpenAI クライアントで自動実行し、CIへ接続する。
